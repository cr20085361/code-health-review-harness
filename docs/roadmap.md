# Roadmap

## 0.1.x

- 建立全局 skill 包。
- 完成核心维度矩阵和报告模板。
- 提供仓库事实采集和安全命令运行脚本。
- 用真实业务仓库做首轮验收。

## 0.2.x

- 增加可选报告落盘模式。
- 增加历史评分对比模板。
- 增加更多语言栈命令策略：Go、Rust、Java、.NET。
- 增加更细的供应链检查项。

## 0.3.x

- 引入可选 JSON 报告结构，方便后续可视化。
- 增加 GitHub Actions 示例，用于定期运行仓库健康检查。
- 增加团队自定义维度覆盖机制。

## 1.0.0

- 稳定触发语、权限边界、维度矩阵和报告契约。
- 完成至少 3 类项目的真实验证：前后端 Web、纯后端 API、桌面或 CLI 工具。
